# 流光活力 (Glow & Go) - 学生生活管理应用

## 🌟 项目简介

这是一款专为大学生设计的移动端生活管理应用，采用"流光活力"设计风格，提供课程管理、社交互动、学业社区和AI智能助手等功能。

## 🎨 设计风格

### 核心理念
- **名称**: 流光活力 (Glow & Go)
- **感觉**: 像夏日清晨的操场，通透、明亮、有冲劲

### 配色方案
- 🧡 **活力橙** (#FF6B35) - 行动按钮
- 💙 **晴空蓝** (#4FC3F7) - 主背景
- 💚 **薄荷绿** (#66FFCC) - 健康数据
- 💛 **柠檬黄** (#FFD93D) - AI提醒
- 💜 **静谧紫** (#B794F6) - 状态指示

### 设计元素
- 磨砂玻璃效果 (Glassmorphism)
- 大圆角 (28px) 果冻按钮
- 柔和的微渐变卡片
- 丰富的动画效果

## 📱 核心功能

### 1️⃣ 今日·元气中枢 (首页)
- ⏰ 3D质感数字时钟
- 🤖 AI智能提醒气泡
- 📚 今日课程卡片（点击查看详情）
- 💪 运动打卡进度（带跳动动画）
- 🍱 智能饮食推荐
- 📅 本周日程概览
- ✨ AI助手浮动球（一对一服务）

**AI助手功能**:
- 点击浮动球展开全屏对话
- 智能回复课程、运动、饮食建议
- 快捷建议按钮
- 支持语音和图片输入

### 2️⃣ 灵犀·搭子群 (社交页)
- 👥 群聊列表（带状态环）
  - 🟣 静谧紫 = 睡觉中
  - 🟠 活力橙 = 学习中
  - 🟢 薄荷绿 = 活跃
  - 🟡 柠檬黄 = 用餐中
- 💬 点击进入群聊详情
- 🎯 AI智能建议（餐厅推荐、投票功能）

**群聊详情**:
- 实时消息展示
- 语音/图片/表情发送
- 视频/语音通话入口
- 消息气泡动画

### 3️⃣ 智慧·学业社区 (发现页)
- 🔍 糖果风格大搜索框
- 🏷️ 快捷分类标签
- 📌 瀑布流布局帖子
- ❤️ 点赞、评论、分享
- 📖 点击查看帖子详情

**帖子详情**:
- 完整内容展示
- 图片网格展示
- 互动功能（点赞、收藏、分享）
- 评论区

### 4️⃣ 我的 (个人中心)
- 👤 个人信息展示
- 📊 数据统计（学习天数、完成任务、健康分数）
- ⚙️ 功能菜单
  - 课程表
  - 学习记录
  - 健康数据
  - 消息通知
  - 设置

## 🎯 详情页面

### 课程详情
- 📍 地点、时间、教师信息
- 📖 今日课程内容
- 📅 近期课程安排
- 📝 作业任务列表
- 🏆 成绩统计
- ⏰ 提醒设置

### 运动详情
- 🎯 今日目标进度环
- 📊 本周运动统计图
- 🏃 最近运动记录
- 🏅 运动成就徽章
- 🔥 卡路里消耗统计

### 帖子详情
- 👤 作者信息
- 📄 完整内容
- 🖼️ 图片展示
- 💬 评论区
- ❤️ 点赞/收藏

## 🎭 动画效果

- ✨ 页面切换动画
- 🎯 卡片悬浮效果
- 💫 按钮Q弹缩放
- 🌊 脉动光环效果
- 📈 进度条动画
- 🎨 渐变色过渡

## 🔧 技术栈

- **框架**: React 18
- **动画**: Motion (Framer Motion)
- **样式**: Tailwind CSS v4
- **图标**: Lucide React
- **构建**: Vite

## 📐 移动端优化

- 响应式设计
- 触摸优化
- 移动端容器 (max-width: 448px)
- 底部导航栏
- 安全区域适配

## 🚀 特色亮点

1. **沉浸式设计** - 全渐变背景，磨砂玻璃效果
2. **智能AI助手** - 主页专属，一对一服务
3. **状态感知** - 群聊状态环实时显示
4. **流畅动画** - 丰富的Motion动画
5. **完整交互** - 所有卡片可点击查看详情
6. **模块化设计** - 组件化架构，易于扩展

## 📝 使用说明

### 导航
- 点击底部导航栏切换主要页面
- 点击卡片进入详情页
- 详情页顶部左侧返回按钮

### AI助手
- 首页右下角浮动球
- 点击展开全屏对话
- 输入问题获取智能建议

### 社交互动
- 群聊列表点击进入聊天
- AI助手球查看智能推荐
- 帖子点击查看详情

## 🎨 设计理念

整个应用围绕"年轻、活力、智能"展开，通过丰富的色彩、流畅的动画和贴心的AI助手，为大学生打造一个充满活力的数字生活空间。
